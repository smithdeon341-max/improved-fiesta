üöÄGetting Started with Stripe

Step 1: Create a Stripe Account

Visit stripe.com and sign up for a new account. Choose "Business" when asked about your organization type.

After verification, you'll get access to your dashboard with your API keys.

Create Stripe Account

Step 2: Get Your API Keys

In your Stripe Dashboard, go to "Developers" > "API Keys" to find:

  • Publishable Key (starts with pk_) - Used in frontend
  • Secret Key (starts with sk_) - Keep private, use in backend
// Replace these with your actual keys STRIPE_PUBLISHABLE_KEY = "pk_test_YOUR_KEY_HERE" STRIPE_SECRET_KEY = "sk_test_YOUR_KEY_HERE"

Step 3: Enable Stripe Connect

Go to "Settings" > "Business Settings" and enable Stripe Connect to accept payments and manage payouts.

This allows you to create connected accounts for your customers to receive payments.

app.post('/webhook', express.raw({type: 'application/json'}), (req, res) => { const sig = req.headers['stripe-signature']; const event = stripe.webhooks.constructEvent( req.body, sig, 'whsec_YOUR_WEBHOOK_SECRET' ); switch(event.type) { case 'customer.subscription.created': // Update user subscription status break; case 'invoice.payment_succeeded': // Process successful payment break; case 'customer.subscription.deleted': // Handle cancellation break; } res.json({received: true}); });

üè¶Bank Account & Payouts Setup

Connect Bank Account

In your Stripe Dashboard, go to "Settings" > "Bank Accounts" and add your bank details:

  • Routing Number (9 digits)
  • Account Number
  • Account Holder Name
  • Account Type (Checking/Savings)
‚ö†Ô∏è Stripe will deposit two small amounts ($0.30-$0.50) to verify your account. Check your bank statement and verify these amounts in Stripe within 10 days.

Automatic Payouts

Once verified, Stripe automatically transfers funds to your bank account:

  • Default: Every 2 business days
  • Minimum: $1 per payout
  • Fees: 2.9% + 30¬¢ per subscription payment
  • Processing Time: 1-2 business days

üîíSecurity Best Practices

API Keys Security

  • Never expose your Secret Key in frontend code
  • Store keys in environment variables (.env)
  • Rotate keys regularly from Stripe Dashboard
  • Use Restricted API Keys for limited access

PCI Compliance

  • Never store credit card data on your servers
  • Always use Stripe Elements or Payment Element
  • Use HTTPS for all transactions
  • Enable 3D Secure for card verification
‚úì Stripe is PCI DSS Level 1 compliant, meaning your data is encrypted and protected by industry standards.

üåçGoing Live to Production

Pre-Launch Checklist

  • Thoroughly test all payment flows
  • Verify bank account connection
  • Set up webhook endpoints for all events
  • Enable 3D Secure for extra security
  • Switch from test keys to live keys
  • Update all payment URLs to HTTPS
  • Add SSL certificate to domain
  • Set up error logging and monitoring

Switch to Live Keys

In your Stripe Dashboard, toggle "Viewing test data" OFF to access live keys:

// Before going live, replace test keys: OLD: pk_test_xxx ‚Üí NEW: pk_live_xxx OLD: sk_test_xxx ‚Üí NEW: sk_live_xxx
‚ö†Ô∏è Live keys will process real charges. Double-check all configurations before switching!
